매물 카탈로그 작성 (내부용)

고객 화면: view.html 작성 화면: editor.html

여기서 내용 입력 → 고객 공유 링크 생성 누르면 고객용 링크가 만들어져.
(서버 없이 URL에 데이터를 담는 방식이라, 링크가 길어질 수 있어. 보통 3~10건은 문제 없음)

임시저장은 이 컴퓨터 브라우저에만 저장돼. 고객 링크에는 영향 없음.

URL 여러 개 붙여넣기 (자동 매물 생성)

줄바꿈으로 여러 개 URL을 붙여넣고 URL로 매물칸 생성을 누르면, URL 개수만큼 매물 입력칸이 자동으로 생겨.

매물 목록 0건
여기에 고객 공유 링크가 생성돼.

고객에게는 이 링크 하나만 카톡으로 보내면 끝.
